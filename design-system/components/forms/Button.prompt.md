Primary/secondary action button — use for form submits and CTAs; `primary` (solid) is the default, `accent` for terracotta emphasis.

```jsx
<Button variant="primary" onClick={go}>Explore →</Button>
<Button variant="secondary">Run again</Button>
```

Variants: primary (fg-on-bg), accent (terracotta), secondary (bordered), ghost. Set `disabled` for pending states.
